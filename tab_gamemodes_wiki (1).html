À propos

As-tu déjà soupçonné tes amis de passer en mode Créatif sur ton SMP ? Ou voulu exposer un admin louche ? Alors ce mod est fait pour toi !

Présentation : TAB Gamemodes ! Créé par Leboss1200.

Ce mod te permet de voir instantanément le gamemode de chaque joueur directement dans la TAB.

Installation

  • Installe Fabric Loader pour Minecraft 1.21–1.21.4
  • Télécharge Fabric API
  • Place le fichier .jar du mod dans ton dossier mods
  • Lance Minecraft avec le profil Fabric

Fonctionnement

Le mod affiche des balises de gamemode dans la TAB :

  • [S] — Survie
  • [C] — Créatif
  • [A] — Aventure
  • [Sp] — Spectateur

Limitations

Fonctionne uniquement avec Fabric
